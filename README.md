# Installation Process

- clone the project

- go to project directory

- create new database named `simplestock_db` from phpmyadmin

- default db username & password is `root` if you want to change then go inside `includes` directory there is `database_connect.php` you can change db username & password.

- import `stock.sql` which is in project directory
